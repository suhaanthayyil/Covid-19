# COVID-19 Long COVID Prediction Pipeline

Machine learning pipeline for predicting Long COVID (hospitalization / return to hospital) from patient demographics and comorbidities. Built for publication in the *Journal of Student Research*.

## Repo structure

```
covid-longcovid-ml/
├── README.md                 # This file
├── requirements.txt          # Python dependencies
├── run_pipeline.py           # One-shot: load data → train → evaluate → SHAP → save figures
├── data/
│   ├── raw/                  # Put your downloaded CSV here (do not commit data)
│   ├── processed/            # Cleaned data (optional, generated by pipeline)
│   └── README.md             # Where to download the dataset
├── notebooks/
│   ├── 01_EDA.ipynb
│   ├── 02_preprocessing.ipynb
│   ├── 03_model_training.ipynb
│   └── 04_explainability.ipynb
├── src/
│   ├── preprocess.py         # Load, clean, encode, scale, SMOTE
│   ├── train.py              # Train LR, RF, XGBoost, SVM + 5-fold CV
│   ├── evaluate.py           # Metrics, ROC curves, confusion matrix
│   └── explain.py            # SHAP beeswarm, bar, waterfall
├── figures/                  # Output PNGs (300 dpi)
└── results/
    └── metrics_table.csv     # All model scores
```

## How to run

1. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Get real data**
   - Run **`python scripts/download_real_data.py`** — downloads PhysioNet (no login) or Mexico/Kaggle data if you have [Kaggle API](https://github.com/Kaggle/kaggle-api) configured.
   - Or manually place a CSV in `data/raw/` (see [data/README.md](data/README.md)).

3. **Run the full pipeline** (trains all models, saves figures and metrics)
   ```bash
   python run_pipeline.py
   ```
   On the full Mexico dataset (~1M rows), training can take a long time. For a quicker run on a stratified sample of real data, use:
   ```bash
   MAX_TRAIN_SAMPLES=50000 python run_pipeline.py
   ```
   Omit `MAX_TRAIN_SAMPLES` for the full dataset (e.g. overnight run).
   Outputs:
   - `figures/fig1_class_distribution.png` … `fig8_shap_bar.png`
   - `results/metrics_table.csv`
   - Console summary: best model, ROC-AUC, top SHAP feature.

4. **Or run step-by-step in Jupyter**
   - Open `notebooks/01_EDA.ipynb` through `04_explainability.ipynb` in order.

## Dataset

We use the Mexico COVID-19 open dataset (or equivalent with the same columns). Long COVID is defined as: confirmed positive (CLASIFFICATION_FINAL 1–3) and `PATIENT_TYPE == 2` (hospitalized / returned to hospital). See the research plan in the repo and [data/README.md](data/README.md).
